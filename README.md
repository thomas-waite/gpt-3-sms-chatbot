# GPT-3 text chatbot

A GPT-3 chatbot connected to Twilio text messaging

To get started:

1. Clone the repo
2. Run `pip install -r requirements.txt`
3. Serve the flask app and make it accessible over the internet: `make serve_app && make serve_URL`
